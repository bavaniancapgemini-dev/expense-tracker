# 💸 Expense Tracker v12.0

A modern Python-based Expense Tracker built for managing daily expenses, budget tracking, analytics, and financial reporting directly from the terminal.

---

# 🚀 Features

## ✅ Expense Management

* Add expenses
* View all expenses
* Delete expenses
* Edit expenses
* Search expenses

---

## ✅ Financial Reports

* Total expense calculation
* Highest expense
* Average expense
* Category-wise report
* Monthly reports

---

## ✅ Budget System

* Monthly budget tracking
* Remaining balance calculation
* Budget exceeded warning system

---

## ✅ Analytics Dashboard

* Expense statistics
* Category analytics
* Spending insights

---

## ✅ Expense Sorting

Sort expenses by:

* Highest amount
* Lowest amount
* Name A-Z
* Latest date

---

## ✅ Export System

* Export expenses to CSV file

---

## ✅ Backup & Storage

* Automatic expense saving
* Persistent file storage system

---

# 🛠 Technologies Used

* Python 3
* File Handling
* Functions & Modules
* Terminal UI
* Data Analytics Logic
