# Expense Tracker v9.0

A smart AI-powered finance management system built with Python.

## Features

* Add / Edit / Delete Expenses
* Budget Tracker
* Category Reports
* Monthly Reports
* Expense Graphs
* AI Financial Insights
* Multi-user Login System
* Expense Analytics Dashboard
* CSV Export
* Smart Budget Warnings

## Technologies Used

* Python
* Matplotlib
* Colorama

## Run Project

```bash
python main.py
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Version

Current Version: v9.0
